# TextbasedAdventure

## Description

C# Console application based Text adventure, inspired by Zork.

## How to run the game

 Clone the repository :
``` shell
git clone https://github.com/LisaWerner856/TextbasedAdventure.git
```
Navigate to the build folder:
<code> TextbasedAdventure/TextAdventure-DS9/builds</code>

Execute the <code>TextAdventure-DS9</code> file.
